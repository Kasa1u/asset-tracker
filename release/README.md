# 有数资产 v0.1.0

## 使用方法

1. 双击运行 `有数资产.exe`
2. 首次运行会自动创建数据库文件
3. 开始记录您的资产信息

## 功能特性

- ✅ 资产管理：记录物品的购买信息、价格、保修期等
- ✅ 日均成本计算：自动计算每个物品的日均使用成本
- ✅ 心愿清单：记录想要购买的物品
- ✅ 时间轴：按时间顺序展示购买记录
- ✅ 统计分析：多维度数据可视化
- ✅ 图片上传：为物品添加图片
- ✅ 多语言支持：中英文切换

## 系统要求

- Windows 10 或更高版本

## 数据存储

应用数据存储在用户数据目录：
- Windows: `%APPDATA%\com.hanyaoyu.youshu\`

## 注意事项

- 首次运行可能需要几秒钟初始化数据库
- 数据库文件为 `assets.db`，请勿删除
- 建议定期备份该文件
